# [06c_room_screen] 01_基本レイアウト作成

## タスク概要

ルーム画面（`/room/:room_id`）の基本的な3分割レイアウトを構築する。

## 実装内容

### 1. ページコンポーネント作成

`src/app/room/[room_id]/page.tsx`

- 動的ルートパラメータ（room_id）の受け取り
- 基本的なページ構造の定義
- レスポンシブレイアウトの設定

### 2. レイアウト構成

#### デスクトップレイアウト（1200px以上）

- **上部バー**: ルーム情報・接続状況表示エリア
- **左側エリア**: 2Dスペース（800x600px）
- **右側エリア**: リアルタイムチャット（400px幅）
- **下部バー**: 設定・退室コントロール

#### レスポンシブ対応

- **タブレット（768px-1199px）**: 左右分割維持、サイズ調整
- **モバイル（767px以下）**: 上下分割レイアウト

### 3. カラーテーマ適用

- **背景色**: Tailwind gray-50
- **アクセント**: Tailwind blue-600
- **テキスト**: Tailwind gray-900
- **ボーダー**: Tailwind gray-200

### 4. 初期プレースホルダー

各エリアに以下のプレースホルダーを配置：

- 上部バー: "ルーム情報エリア"
- 左側: "2Dスペースエリア"
- 右側: "チャットエリア"
- 下部バー: "設定・退室エリア"

## 成果物

- `src/app/room/[room_id]/page.tsx`
- レスポンシブ対応した3分割レイアウト
- 基本的なスタイリング
